# Changelog

## 1.0.1 — Packaging fix

- Exclude compiled Python cache files (`python/**/__pycache__/`) from the

## 1.0.0 — Initial public release

First public open-source release of festo-codesys-mcp: an MCP server that generates
IEC 61131-3 Structured Text and PLCopen XML for Festo/CODESYS PLCs, with an embedded
knowledge base, reusable Function Block library, and validation pipeline.
